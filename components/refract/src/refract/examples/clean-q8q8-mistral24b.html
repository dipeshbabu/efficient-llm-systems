What is this? REFRACT — REFerence-anchored Robust Acid-test for Compressed Transformers — is a quantization audit framework that scores how faithfully a quantized KV-cache config preserves the model's own fp16 behaviour. The repository contains documentation, the motivation paper, and guidance on interpreting these scores.
