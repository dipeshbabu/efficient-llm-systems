# Research record

This directory preserves the evidence and reasoning behind Efficient LLM
Systems. Documents are dated snapshots: they may describe failed approaches,
superseded implementations, or conclusions that apply only to a specific
model and hardware matrix.

## Sections

- [`papers/`](papers/) contains consolidated research reports and major
  findings.
- [`investigations/`](investigations/) contains experiment logs, validation
  studies, performance analysis, and engineering investigations.
- [`archive/`](archive/) contains superseded plans, the original implementation
  plan, and historical documentation.

Use the [documentation index](../docs/index.md) for current operational
guidance. Raw outputs referenced by these reports live in
[`artifacts/`](../artifacts/README.md).

Negative results are intentionally retained. Treat every performance or
quality claim as scoped to the model, engine, configuration, context length,
hardware, and date recorded by its source document.
